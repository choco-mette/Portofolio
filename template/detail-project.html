Terminal Lo-Fi Player

Aplikasi CLI streaming audio ringan yang dirancang untuk developer yang ingin mendengarkan musik santai tanpa perlu membuka browser atau GUI berat.

Source Code ↗ Live Demo ⚡

Masalah & Solusi

Saat melakukan deployment server atau mengonfigurasi docker-compose, sering kali kita harus fokus sepenuhnya pada terminal. Membuka tab Chrome atau aplikasi Spotify sering kali mengganggu workflow dan memakan RAM (terutama saat membuka banyak container).

Sebagai solusinya, saya merakit Terminal Lo-Fi—sebuah pemutar musik berbasis Bash dan Python yang menarik stream audio ringan langsung dari stasiun radio Lofi Girl, lengkap dengan UI di terminal menggunakan ncurses.

"Musik yang bagus untuk coding seharusnya terdengar di latar belakang, bukan menyedot resource sistem di latar depan."

Arsitektur Eksekusi

Aplikasi ini menggunakan modul Python vlc-bindings untuk menangani output audio agar kompatibel dengan PulseAudio di Linux, sementara Bash menangani visualisasi UI dan input perintah (seperti pause/play/skip).

player.py 
import vlc
import time

# Initialize standard VLC instance
instance = vlc.Instance("--no-video")
player = instance.media_player_new()

def play_stream(url):
    media = instance.media_new(url)
    player.set_media(media)
    player.play()
    
    while True:
        pass # Keep script alive
